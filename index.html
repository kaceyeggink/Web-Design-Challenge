Project Summary:Atmospheric Measurements by Latitude

The purpose of this project was to analyze weather changes based on thier distance from the equator. The data from 500 cities was pulled from the OpenWeatherMap API. Matplotlib was utilized in order to create visulizations for changes in temperature, cloudiness, wind speed, and humidity. The visualizations, comparisons and raw data set are included in this website.
